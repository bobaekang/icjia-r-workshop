Importing/Exporting Data

Source: Wikimedia.org

Comma-separated values (.csv)

  • readr package (tidyverse)
    • read_csv()
    • write_csv()
  • data.table package
    • fread()
    • fwrite()

Excel spreadsheets (.xlsx/.xls)

  • readxl package (tidyverse)
    • read_excel()
    • write_excel()
  • readxl on tidyverse.org
  • readxl CRAN documentation
  • readxl Github repository

SPSS data files (.sav)

  • haven package (tidyverse)
    • read_sav()
    • write_sav()
  • haven also has functions to import/export the file formats of other statistical softwares
    • STATA
    • SAS
  • haven on tidyverse.org
  • haven CRAN documentation
  • haven Github repository

A “fast-on-disk” data frame storage (.feather)

  • feather package (tidyverse)
    • read_feather()
    • write_feather()
  • The .feather format is also supported in Python!
  • feather on tidyverse.org
  • feather CRAN documentation
